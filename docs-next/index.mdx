# What is ErsatzTV Next

<span style={{ fontSize: "1.4rem" }}>
  ErsatzTV Next is a modular, self-hosted IPTV server that transcodes and
  streams your media as live TV channels.
</span>

ErsatzTV Next aims to provide a lighter alternative to all-in-one systems like [ErsatzTV Legacy](https://ersatztv.org/docs/) by focusing on transcoding and streaming, supported by third-party library management and scheduling tools. A basic scheduler is included for demonstration and reference purposes.

## Quickstart

ErsatzTV Next is supported on most platforms and operating systems.
